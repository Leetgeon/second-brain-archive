# Second Brain Archive 이용약관 초안

최종 업데이트: 2026년 6월 14일

## YouTube 서비스

Google 계정 연결 기능은 YouTube API Services를 사용한다. 사용자는 앱을
사용함으로써 [YouTube 이용약관](https://www.youtube.com/t/terms)과
[Google 개인정보처리방침](https://policies.google.com/privacy)의 적용에
동의한다.

## 공개 배포 모드

공개 배포 모드는 YouTube 재생목록의 제목, 항목과 원본 링크를 사용자의 로컬
카탈로그에 표시한다. YouTube 영상, 음성 또는 자막의 다운로드, 백업 및 오프라인
재생 기능은 제공하지 않는다.

## 로컬 파일

사용자는 자신이 소유하거나 적법하게 사용할 수 있는 로컬 파일만 가져와야 한다.
앱은 사용자가 제공한 자료의 저작권이나 소유권을 취득하지 않는다.

## 계정 연결 해제

사용자는 앱 또는 Google 계정 보안 설정에서 언제든 접근 권한을 철회할 수 있다.
앱에서 연결을 해제하면 로컬 토큰과 Google 권한으로 가져온 카탈로그 항목이
삭제된다.
